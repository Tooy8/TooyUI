# 安装

## 环境支持

由于 Vue 3 不再支持 IE11，TooyUI 也不再支持 IE11 浏览器。

## 版本

0.03

## 使用包管理器

建议您使用包管理器 (NPM, Yarn, pnpm) 安装 TooyUI, 然后您就可以使用打包工具，例如 Vite 和 webpack




### 选择一个你喜欢的包管理器

### NPM

```vue
$ npm install tooy-ui --save
```

### Yarn
```vue
$ yarn add tooy-ui
```

### pnpm
```vue
$ pnpm install tooy-ui
```
